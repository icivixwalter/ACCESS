Funzionalità macro di Excel 4.0

Il file della Guida in linea relativo alle funzionalità macro di Microsoft Excel 4.0 contiene le informazioni di riferimento per tutte le funzionalità macro di Excel 4.0 (macro XLM) disponibili in Excel. È possibile scaricare il file della Guida relativo alle funzionalità macro dal sito Web Microsoft Office Online.

Protezione  Le macro possono contenere virus, pertanto è consigliabile eseguirle con cautela ed eseguire solo quelle provenienti sicuramente da un'origine affidabile. Assicurarsi inoltre che nel computer sia installato un programma antivirus aggiornato. Si noti che a differenza di altre macro, le macro XML non possono contenere una firma digitale. Inoltre non è possibile aprire una cartella di lavoro contenente macro XLM se il livello di protezione per le macro è impostato su Elevata.
